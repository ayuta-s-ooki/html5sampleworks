#ContentEditable

contenteditable属性を使うと、属性をセットされている要素内のコンテンツを編集することが可能になります。

##対応状況

[contenteditable](http://caniuse.com/#feat=contenteditable)

##サンプル

`contenteditable`属性がセットされている要素を、自由に編集・変更することができます。
ローカルストレージも使い変更内容を保存しています。
